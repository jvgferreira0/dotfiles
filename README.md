# Dotfiles Repository

> **Note:** This repository is now archived in favor of the Baal framework.

## Migration to Baal

The configurations in this repo have been migrated to the **Baal dotfiles framework** located at:
- `~/config/baal/` (or `~/.config/baal/`)

## Current Status

- `zshrc` - Archived (now using Baal's modular config)
- Shell now loads from: `~/.config/baal/defaults/zsh/rc`

## Baal Framework

Baal provides:
- Modular configuration (split into logical files)
- Theme support (easy color scheme switching)
- Better organization and maintainability

See: `~/config/baal/README.md` for more info

This repo is kept for reference and history.
